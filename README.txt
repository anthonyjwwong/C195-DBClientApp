Scheduling App GUI created to help a company track the customers and appointments.

## === Software Information ===

JavAFX version: JavaFX -20
MySQL Connector Driver Version: mysql-connector-j-8.0.33

## === Directions ===
1. Launch the application
2. Login with the valid username and password 
3. Go to either the appointment or customer main page
4. Add or update according to your needs.
5. Go to report page to check reports.
6. Exit when you're done.

## === Reports === 
A report regarding the login will be generated on login_activity.txt at the root folder of the project. 
The report will show the user, login time/date.
